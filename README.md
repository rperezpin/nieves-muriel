# Landing Page — Nieves Muriel García

Sitio web personal tipo landing page para **Nieves Muriel García**, escritora, poeta e investigadora, profesora en la Universidad de Málaga.

## Descripción

Landing page estática y minimalista que presenta el perfil profesional y literario de Nieves Muriel García. El diseño utiliza una paleta de colores en tonos turquesa y está pensado para ser elegante, limpio y fácil de navegar.

## Estructura del sitio

```
index.html          → Página principal (single page)
assets/
  css/
    styles.css      → Estilos principales
  img/
    foto-nieves.jpg → Foto de perfil (a proporcionar)
    placeholder.svg → Placeholder mientras no haya foto real
```

## Secciones

1. **Hero / Sobre ella** — Foto de perfil con una breve biografía
2. **Libros publicados** — Listado de sus obras con descripción breve
3. **Editoriales** — Enlaces a sus perfiles en editoriales
4. **Colaboraciones** — Enlaces a colaboraciones, artículos y apariciones

## Datos de la autora

### Biografía resumida

Nieves Muriel García (Melilla, 1977). Doctora en Estudios de las Mujeres por la Universidad de Granada. Licenciada en Filología Hispánica. Máster en Estudios de la Diferencia Sexual (Universitat de Barcelona). Profesora en la Universidad de Málaga. Especialista en escritura femenina y poesía española contemporánea. Premio Beca de Investigación Miguel Fernández (2013). Premio de Ensayo Feminista María Telo (2016). Ganadora del II Premio de Poesía Juana Castro.
